### 🏃 Run locally

Would like to run this project locally? Open terminal and follow these steps:

1. Clone the repo
   ```sh
   git clone hhttps://rokasime.github.io/54-grupe-ufo/
   ```
2. Install NPM packages
   ```sh
   npm i
   ```
   or
   ```sh
   npm install
   ```
3. Run the server
   ```sh
   npm run dev
   ```
